 通过springboot整合mybatis，代码涉及到两种实现方式，一种是传统的xml文件的形式；
 另外一种是springboot自动配置的注解实现方式。
 》项目中还是用到了阿里的Druid数据源，方便数据后端监控。
## springboot_jsp 整合jsp

 是如何实现。

 使用spring缓存，通过与springboot相结合，使用注解的方式，来实现缓存。
